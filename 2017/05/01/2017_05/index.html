2017.05笔记

“我想自己终究是幸运的，不仅仅是因为那些外在的所得，而且因为我还挺结实的。总是被打得七零八落，但总还能在上帝他老人家数到“九”之前重新站起来，再看到眼前那个大海时，还是一样兴奋，欢天喜地地跳进去。在辽阔的世界面前，一个人有多谦卑，他就有多快乐。”

——《一个人要像一支队伍》，刘瑜

我这几个月在做什么

刚刚决定写个博客之前去看了一月份写的那篇，上面说今年的小目标是要去个外企，哈哈哈，有点伤感啊，因为我感觉去不了了。

自从二月返校以后，我一直投身于找实习的大军中，现在结果不是很理想，先记录一下自己的心路历程吧。

腾讯

最先开始的是腾讯。

老实讲，我对自己的简历是有信心的，腾讯的简历被评为S，这让我自我感觉良好，而且一开始是想去外企的，所以好多小公司都没有投，这奠定了悲惨的故事走向。

还是先说腾讯，很快接到电话面试，一是ES6不熟，二是我几乎没有用过框架，三是实习经历里没有做过对前端有创造性的东西，卒。

在我意识到自己存在这些问题之后，我是很想努力的把差的东西补上来的，然后开始研究框架，但是，我实在是太忙了，一半的时间在面试，一半的时候在写实验室的代码，还要投简历填表格，唉。

好吧其实上面都是借口，就是我水平不行看起来比较吃力。

虽然内推的轮次挂了，但是我没有放弃，我准备去参加笔试。

下面我给大家讲个笑话。

那一天是晚上七点笔试，我约好了一起答题的小伙伴，五点我从实验室回到宿舍，六点吃完饭，小伙伴还没准备好，我就拿出来一个鸭梨，咬了第一口，

直接就咽了

没给我卡死

于是我就去了校医院，大夫说让我喝醋，我觉得非常不靠谱，决定先去答笔试，笔试全程满脸通红被卡的说不出话来，答应和我一起答题的小伙伴也弃我于不顾，于是我就随便写了写，打车去北医三院了。

于是笔试也挂了。说实话我现在写下这段读一遍都觉得要卡死了。

但是我还是没有放弃，我去参加了霸面。

看到前端的简历有一枚一元硬币立起来那么厚的时候，我就知道我肯定没有面试机会了，于是我默默的把简历放在了上面回学校了。

那一刻我为数不多的意识到了生活的艰难。

希望与腾讯校招再见。

亚马逊

面试挂了，但是ss学长特别靠谱，一直帮我争取各种机会，我们甚至未曾谋面但学长却如此热情的帮助我，真的很感谢，还好实验室有很多小伙伴去了亚马逊，希望以后可以通过他们认识一下学长，有幸能请他吃个饭就再好不过啦。

真的很喜欢亚马逊的工作氛围，在找不到工作的时段里我有无数个瞬间想给原来的mentor发微信问问我还能不能回去，然而还是忍住了。

哈哈，有缘再见。

阿里

阿里的内推非常的曲折，我投的本来是阿里云，面了四面之后杳无音信，又被蚂蚁，还是杭州的蚂蚁录取了，我现在还拿着offer不知道要不要点接受。

神烦。
