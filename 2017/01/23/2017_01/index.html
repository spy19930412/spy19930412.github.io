2017.01笔记

“过去的如果就这么过去了，往后只会越来越糟”——《驴得水》

总结我的2016年

“过去的如果就这么过去了，往后只会越来越糟”——《驴得水》
——题记

总的来说2016年还不错哦，2月给自己动了一刀，4月中生日那天去拔了智齿，五月去了趟香港，六月去阿里，九月去亚马逊，12月离职之后去泰国浪了一波，嘿嘿。

泰国旅游

回忆类似于从栈顶取数据，只能想起来离现在最近的泰国游了。一行五人，七天去了五个地方，玩儿的非常愉快，泰国的旅游业是用心当成事业来做的，让你时刻感受到这地不错以后没准可以带别人一起来，不是在国内某景区那种，到处人多，没地可去之后选个地，去了之后想回家哭的那种。

1.上海
在上海只短暂的待了半天，去吃了小笼包肠粉一类之后去逛了外滩。晚上看了外滩夜景之后去坐了磁悬浮，这种从市区到机场只要不到半小时的感觉真的很爽，然而穷逼如我真的贵哭。

2.泰国
在泰国的时候空气很好，真的每一天都觉得是那种“蓝蓝的天空·绿绿的海水”，去了三天海边，每天坐在船上从流飘荡，任意东西，觉得自己很有浪漫主义情怀。

在泰国最让我糟心的是语言，因为我们住的地方太多中国游客，所以当地的销售好多都会说英语和汉语，于是就是我们说英语他们全懂，我们说汉语他们基本懂，但是他们说的泰语我们完全不懂。

跨年那天，我们和西班牙人英国人一起在旅店趴踢，晚上在Patong beach上和好多外国人一起玩耍。要说的是，走的时候海滩上太多废弃啤酒瓶子了，不乏很多体面的人。

上学的时候老师说君子慎独，我的理解是君子在自己一个人的时候也会按照周围有别人的时候那样要求自己，与环境无关；比如在空无一人的马路上，红灯，要不要过去的问题。

扔废弃啤酒瓶子到海滩上的歪果仁总不能抱怨周围人都在扔，自己不得不扔吧。

这个问题告诉我：

  1. 不要把人想的太好
  2. 不要把人想的太坏

亚马逊实习

在外企特别好的地方是可以看到很多的软实力，大家工作上其实真心不是很卖命，但是真的是很优秀的人啊。
今年有两个小目标，一是希望找到一份外企的工作，二是尽早考过雅思
希望能早日看到更大的世界

一点废话

人生时混沌，终时亦为一无所知，人们说要活的精彩，如何才算活的精彩呢。

世界就像一个生态系统，人们的成就基本上是一个正态分布的形态，从事的工作的于我来说虽然千差万别，但毫无优劣之分，从宇宙大爆炸开始，生命不过是一个熵减到熵增的过程。始于尘土，终于尘土。

如此说来，那什么才叫精彩。

于我而言，在我有生之年能够通过自己的努力上升一个阶级，才叫精彩。

reference

热力学第二定律 即 熵增定律。
内容是 在无“有意识的”外力干预下 任何系统的内部 都只会向无序的方向进展 最终变成一团完全混乱无序而平等的“原子形成的浓汤而已”。
假设 人类没有灵魂 没有神 那么根据熵增定律 不要讲人类 就是生命 都无法产生。
退一万步 假设生命最早真的起源于病毒这样最基本的RNA单链。学过化学的人都知道 有机物不同于无机物的原子之间的化学键。有机物的碳氢氮原子之间 结合与拆离时 所需要的力有多强大。
因此 漫游的原子与原子之间 要形成有机分子 特别是遗传物质 在没有高级生命才具备的由一系列复杂的蛋白与辅酶形成的“氢能量传递链”时 需要多巨大的能量轰击才能稳定的聚在一起。
当时地球的条件 只有闪电的能量 能达到。
这就意味着 一种病毒要变成另外一种病毒 而不是变种 就必须需要来一次闪电轰击 才能拆掉原子 或 拼接新原子。
一个普通蛋白与糖体 光涉及的主碳链就上万个 合成都那么难 何况是 多达千亿个原子的遗传物质？！
遗传物质在自然界 只能复制上代而产生 绝对无法从无机物原子合成出来的先例。
因为 其需要的能量 不是人类能够控制的巨大。

这一冲突 至今 进化论者无法完美解释。
甚至连“中间过渡化石”都至今没有被发现。
在欧美国家 进化论作为一种学说 仍是科学界的主流 但是 只是作为一种学术假说 在论文界流行。
一下班科学家跑教堂的还是多数
