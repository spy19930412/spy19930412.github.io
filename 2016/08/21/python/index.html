什么是数据结构，什么是数据类型

int char float double 这些是 数据类型 ，类比作原子，他们间的不同就是元素种类的不同，只涉及自身的不同，比如两种原子内的质子中子数不同，是一种内在属性。

链表 队列 堆栈 树 这些是数据结构，类比作分子，他们间描述的是数据间的关系，就如同分子描述了原子的组合方式
