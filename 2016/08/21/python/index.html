    set 和 dict 类似，也是一组 key 的集合，但不存储 value。由于 key 不能重复，所以，在 set 中，没有重复的 key。

    通过add(key)方法可以添加元素到 set 中，可以重复添加，但不会有效果

    通过remove(key)方法可以删除元素
